# Installation

## Stable release

To install xtclass, run this command in your terminal:

```sh
uv add xtclass
```

Or if you prefer to use `pip`:

```sh
pip install xtclass
```

## From source

The source files for xtclass can be downloaded from the [Github repo](https://github.com/sandorn/xtclass).

You can either clone the public repository:

```sh
git clone git://github.com/sandorn/xtclass
```

Or download the [tarball](https://github.com/sandorn/xtclass/tarball/master):

```sh
curl -OJL https://github.com/sandorn/xtclass/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd xtclass
uv pip install .
```
